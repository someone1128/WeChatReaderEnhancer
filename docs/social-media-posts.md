## 知乎

```
# 分享我开发的一款微信公众号阅读增强插件

作为一名程序员，我经常需要在电脑上阅读微信公众号的技术文章。但长期以来，我一直被这几个问题困扰：

1. 长文章没有目录，找不到想看的内容
2. 不知道文章结构，读着读着就迷失了
3. 图片太小看不清细节
4. 滚动过程中不知道自己读到哪里了

为了解决这些问题，我开发了一款名为"公众号阅读增强插件"的Chrome/Edge浏览器扩展。

## 核心功能

- **文章目录导航**：自动生成文章结构大纲，点击即可跳转
- **图片查看增强**：点击图片可放大查看
- **阅读位置追踪**：滚动时自动高亮当前阅读位置
- **界面定制优化**：简洁设计，不影响原有阅读体验

## 技术实现

插件采用TypeScript开发，遵循Chrome扩展的Manifest V3规范。主要实现原理是：

1. 分析DOM结构，提取文章中的标题元素
2. 构建树状结构的目录数据
3. 创建目录UI组件并注入页面
4. 监听滚动事件实时更新高亮状态
5. 为图片元素添加点击事件实现放大功能

## 开源与获取

插件已在GitHub开源：[GitHub仓库地址]
可在Chrome商店/Edge商店搜索"公众号阅读增强插件"下载安装

欢迎试用，也欢迎提出改进建议！
```
